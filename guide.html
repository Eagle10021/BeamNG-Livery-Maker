Troubleshooting: Format Errors

If you see an error saying a Template or Skin is unsupported, it's likely using BC7 (DX10) compression.

Common "Unsupported Format" system alert.

Why BC7 is Blocked

Modern vehicles use BC7 compression to save space, but it's too complex for browser-based tool decoding. Use these supported formats instead:

DXT5 (BC3) Livery standard (Color + Alpha)
DXT1 (BC1) Opaque maps (No Transparency)
BC4 / BC5 Grayscale & Normal Maps
RGBA Uncompressed Lossless (Master)

How to Fix: Convert to DXT5

Convert your BC7 templates before loading them into this tool:

A

Method 1: Photopea

  1. Open BC7 file in Photopea.
  2. Go to File > Export As > More > DDS.
  3. Select DXT5 and click Save.
B

Method 2: Standalone

Use ddsconverter.com for batching.

Or the Intel Texture Works plugin for Photoshop.

âœ… Pro Tip: Converting to DXT5 (BC3) restores visual fidelity in the editor and ensures perfect game compatibility.
